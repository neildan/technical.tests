## Requisitos:

## Correr Proyecto:

## Opción 1 (Si docker):
Posicionarse en el proyecto y ejecutar los siguientes comandos:
1. `docker build -t ibm-test .`
2. `docker run --publish 3000:3000 ibm-test`

## Opción 2 (No docker):
Posicionarse en el proyecto y ejecutar los siguientes comandos:
1. `npm i`
2. `npm start`
